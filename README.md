# ☀️ SunCart – Summer Essentials Store

A modern summer eCommerce platform to explore and purchase seasonal products.

## 🌟 Features
- Browse summer products (sunglasses, outfits, skincare, sandals, accessories)
- User authentication with email/password and Google OAuth (BetterAuth)
- Protected product detail pages
- My Profile with update capability
- Responsive design with summer theme
- Smooth animations with Animate.css

## 🛠️ Tech Stack
- **Next.js** (App Router)
- **Tailwind CSS** + **DaisyUI**
- **BetterAuth** (authentication)
- **Animate.css** (animations)

## 🚀 Installation
```bash
git clone https://github.com/YOUR_USERNAME/suncart
cd suncart
npm install
```

Create `.env.local`:
```env
BETTER_AUTH_SECRET=your-secret
BETTER_AUTH_URL=http://localhost:3000
NEXT_PUBLIC_BETTER_AUTH_URL=http://localhost:3000
GOOGLE_CLIENT_ID=your-google-client-id
GOOGLE_CLIENT_SECRET=your-google-client-secret
```

```bash
npm run dev
```
